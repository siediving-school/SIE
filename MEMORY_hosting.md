# Hosting: GitHub Pages
Deploy automático via GitHub Actions al hacer push a `main`.
- Repo: https://github.com/siediving-school/SIE.git
- Workflow: .github/workflows/deploy.yml
- Comando: git push origin main → build + deploy automático (~2 min)
- El sandbox NO puede hacer git commit/push (sin permisos en .git y sin credenciales GitHub)
- El usuario debe correr los comandos git desde su Terminal local
